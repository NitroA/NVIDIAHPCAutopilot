# Container management

	## Start container
	
	docker start CONTAINER ID
	
	## Stop container
	
	docker stop CONTAINER ID
	
	## Exit container without stopping it

	Ctrl + P + Q

	## Return to alive container

	docker attach CONTAINER ID
